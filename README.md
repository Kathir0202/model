This repository contains HTML and CSS files that render a multi-page restaurant website including:

- 📍 Landing (Home) page
- 🖼 Gallery page
- 🧾 Booking form
- 📁 Styles and assets

---

## 🗂 Project Structure

model/
├── images/                  # Image assets for the website
├── booking.html             # Booking page UI
├── gallery.html             # Gallery display page
├── index.html               # Main landing page
├── style.css                # Global site styles
├── gallery.css              # Gallery page styles
├── booking.css              # Booking page styles
└── README.md

---

## 🧠 Key Features

- Semantic HTML5 layout
- Modular CSS for separate pages
- Simple responsive UI suitable for restaurant showcase
- Static website — no backend dependencies

---

##  Pages Included

### Home (`index.html`)
Main landing interface for KT-Foods.

### Gallery (`gallery.html`)
Display of food images and related content.

### Booking (`booking.html`)
UI form for table/booking interaction (frontend only).

---

##  Technologies Used

- HTML5 — Page structure & content
- CSS3 — Styling for each page
- Static Assets — Images stored in images/

---

##  Notes

- This repository contains frontend assets only.
- Forms (like booking) are present purely at the UI level (no backend logic).
